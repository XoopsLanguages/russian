Help: Сontent

Description

Content is a module for adding static pages to Xoops. It could also be used to create a main menu fro the site.

Install/uninstall

No special measures necessary, follow the standard installation process – extract the "сontent" folder into the ../modules directory. Install the module through Admin -> System Module -> Modules. If you need detailed instructions on how to install a module, please see the XOOPS Operations Manual.

Bugs

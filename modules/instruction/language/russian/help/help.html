ОПИСАНИЕ


Модуль Instruction - это решение для публикации на вашем сайте XOOPS

Установка/Удаление

Никаких специальных мер не требуется, следуйте стандартным процедурам установки – извлеките папку модуля в ../modules директорию. Установите модуль через Админ -> Модули системы -> Модули.

Подробные инструкции по установке модулей доступны в XOOPS Operations Manual

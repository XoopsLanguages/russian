Установка XOOPS

Первоначальная установка

Предисловие:

XOOPS — это система веб-публикаций с открытым исходным кодом, объектно-ориентированная, написанная на PHP. Это идеальный инструмент для разработки как малых, так и крупных динамических веб-сайтов сообществ, внутрикорпоративных порталов, корпоративных порталов, веб-журналов и многого другого.

XOOPS выпущен на условиях GNU General Public License (GPL) версии 2 или более поздней и является бесплатным для использования и изменения. Он бесплатен для распространения при условии соблюдения вами условий распространения GPL.

Требования

  • WWW-сервер (Apache, NGINX, IIS и т. д.)
  • PHP 7.4 или выше, рекомендуется 8.3.0+
  • MySQL 5.7 или выше, рекомендуется MySQL 8.4.5+

Перед установкой

  1. Правильно настройте WWW-сервер, PHP и сервер базы данных.
  2. Подготовьте базу данных для вашего сайта XOOPS.
  3. Подготовьте учетную запись пользователя и предоставьте пользователю доступ к базе данных.
  4. Сделайте каталоги и файлы доступными для записи: %s
  5. Из соображений безопасности настоятельно рекомендуется переместить два каталога ниже из корневого каталога документов и изменить имена папок: %s
  6. Создайте (если их еще нет) и сделайте каталоги доступными для записи: %s
  7. Включите файлы cookie и JavaScript в вашем браузере.

Установка локально

Если вы запускаете локальную среду для разработки или тестирования, убедитесь, что соблюдены предыдущие требования. Как только это будет сделано, скопируйте содержимое каталога htdocs (из дистрибутива XOOPS) в корневой путь документа вашей веб-среды. После того как файлы будут скопированы туда, вы можете начать установку, направив свой браузер на "http://вашсайт.com". Это запустит процесс установки.

Установка на хостинговой платформе

Если вы запускаете установку в среде хостинга, распакуйте файлы XOOPS локально или на сервере, если у вас есть доступ к командной оболочке (например, SSH). После этого обязательно переместите или скопируйте все файлы XOOPS из каталога htdocs в ваш корневой веб-каталог (ваш провайдер обычно предоставляет это местоположение с инструкциями). После того как файлы будут скопированы туда, вы можете начать установку, направив свой браузер на "http://вашсайт.com". Это запустит процесс установки.

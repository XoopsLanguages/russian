Для данных XOOPS

1 Из соображений безопасности настоятельно рекомендуется переместить эту папку за пределы корневой директории документа (Document Root).

2 Сделайте эту папку доступной для записи.

3 Убедитесь, что следующие подпапки существуют и доступны для записи:
  configs/
  configs/captcha/
  configs/textsanizer/
  caches/
  caches/xoops_cache/
  caches/smarty_cache/
  caches/smarty_compile/
  data/
  protector/

Примечание: Эти директории уже должны существовать и должны быть скопированы из архива установки. В этих папках содержатся критически важные файлы.

Для получения полной документации по установке или обновлению XOOPS, пожалуйста, посетите:
https://xoops.gitbook.io/xoops-install-upgrade/

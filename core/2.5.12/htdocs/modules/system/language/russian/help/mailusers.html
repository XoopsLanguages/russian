Помощь: Отправка писем пользователям

Описание

XOOPS имеет встроенную систему обмена сообщениями, которая позволяет отправлять сообщения всей базе зарегистрированных пользователей, отдельным подгруппам или с использованием фильтров по дате. Сообщение может быть отправлено в виде личного сообщения (через встроенную систему обмена сообщениями XOOPS) или по электронной почте. Доступен ряд опций для фильтрации подгрупп пользователей из вашей базы.

Также вы можете использовать заранее определенные теги, которые позволят персонализировать ваше сообщение.

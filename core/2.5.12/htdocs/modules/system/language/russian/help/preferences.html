Справка: Настройки

Описание

Панель настроек позволяет вам управлять глобальными параметрами сайта. Стоит внимательно пройтись по ним, так как некоторые параметры могут значительно повлиять на производительность, безопасность и удобство использования сайта.

Общие настройки

Это самые важные и часто используемые параметры. Здесь вы укажете название сайта и слоган. Также выберете тему оформления по умолчанию и интерфейс админки. При необходимости вы можете временно закрыть сайт для всех пользователей. Также можно задать время кэширования на уровне модулей.

Мета-теги и нижний колонтитул

Мета-теги содержат информацию о вашем сайте — они описывают его назначение и помогают поисковым системам правильно индексировать страницы. Эти данные добавляются в заголовок каждой страницы. Они должны отражать общее содержание сайта, так как одинаковы для всех страниц и в данный момент не могут быть настроены индивидуально для модулей или отдельных страниц.

Учитывайте, что разные поисковые системы используют разные методы индексирования. Некоторые учитывают мета-теги, другие — игнорируют. Поэтому стоит изучить, как работают основные поисковые системы, и соответствующим образом оптимизировать сайт и мета-информацию.

Старайтесь быть точными и честными при заполнении мета-тегов. Не пытайтесь манипулировать результатами — поисковики могут распознать это и понизить рейтинг вашего сайта или исключить его из поиска.

Поля Meta Copyright и Footer поддерживают два тега подстановки для упрощения обслуживания:
 - {X_SITEURL} будет заменён на полный URL главной страницы сайта.
 - {X_YEAR} будет заменён на текущий четырёхзначный год, избавляя от необходимости вручную обновлять дату.

Настройки фильтрации слов

XOOPS может автоматически фильтровать нежелательные слова, публикуемые пользователями, и заменять их на заданное вами выражение. Эта функция требует ресурсов сервера и может немного замедлить сайт, поэтому по умолчанию она отключена.

Чтобы включить фильтрацию, выберите опцию «включить». В поле «слова для фильтрации» укажите список слов, разделённых вертикальной чертой (|). Фильтрация нечувствительна к регистру. В поле «заменить плохие слова на» введите текст для замены. По умолчанию — #OOPS#.

Настройки поиска

Поиск — один из важнейших инструментов для портального сайта. В XOOPS результаты поиска группируются по модулям: новости, загрузки, ссылки и т.д., в зависимости от установленных модулей.

Глобальный поиск по всему сайту включён по умолчанию. Если вы отключите его, блок поиска перестанет работать. Минимальная длина ключевого слова по умолчанию — 5 символов. Более короткие слова игнорируются, что помогает повысить точность, но может помешать при поиске фраз вроде «карта сайта».

Поиск требует значительных вычислительных ресурсов, особенно при коротких запросах, которые встречаются чаще и возвращают больше нерелевантных результатов. Исключение коротких слов снижает нагрузку на сервер. Однако в некоторых случаях возможность поиска по коротким словам бывает полезной.

Настройка Email

В этом разделе вы настраиваете параметры электронной почты для системы. Рекомендуется оставить способ отправки по умолчанию — «PHP (mail)». Поля Sendmail или SMTP необходимо заполнять только при использовании альтернативных методов отправки почты.

Параметры аутентификации

Оставьте параметр по умолчанию — база данных XOOPS, и игнорируйте остальные поля.

В систему добавлена поддержка серверов LDAP и Microsoft Active Directory в качестве альтернативы обычной системе пользователей XOOPS. Это удобно для компаний, в которых уже существует база пользователей в ADS или LDAP — такие пользователи смогут входить на сайт без повторной регистрации.

Настройки системного модуля

Здесь вы можете включить или отключить различные разделы системного модуля, а также указать глобальные параметры, например количество отображаемых аватаров или изображений на странице. Также можно выбрать редактор по умолчанию для блоков, комментариев и модулей.

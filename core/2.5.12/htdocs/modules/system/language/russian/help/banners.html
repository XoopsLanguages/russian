Помощь: Баннеры

Описание

Баннеры — это рекламные объявления, отображаемые в верхней части страницы. Они могут быть изображениями или флэш-роликами, например, яркий баннер XOOPS, который входит в стандартную установку. В XOOPS встроена система для управления баннерами и создания учетных записей для рекламных клиентов, которая находится в Модуле системы -> Баннеры. На сайте XOOPS можно использовать несколько баннеров — область отображения поочередно показывает каждый из активных баннеров при загрузке новой страницы.

Каждый раз, когда баннер показывается посетителю, это засчитывается как одно «впечатление», что является стандартной единицей измерения в торговле баннерами. Клиенты обычно платят за определенное количество впечатлений, то есть за то, чтобы их реклама была показана определенное количество раз.

Обратите внимание: Если вы занимаетесь продажей баннеров на основе количества впечатлений, есть одна важная вещь, которую следует сделать: Перейдите в Модуль системы -> Настройки -> Общие настройки и заполните поле «Ваш IP-адрес» (если он у вас есть). Это предотвратит учет ваших посещений сайта в подсчете впечатлений. Непринятие этой меры может считаться несколько нечестным (хотя стоит отметить, что многие администраторы не имеют постоянного IP-адреса, например, те, кто использует модемы с dial-up подключением).

Справка: Личные сообщения

ОПИСАНИЕ

Модуль ЛС предназначен для обмена личными сообщениями между пользователями

УСТАНОВКА/УДАЛЕНИЕ

Никаких специальных мер не требуется, следуйте стандартному процессу установки – извлеките папку /pm в каталог ../modules. Установите модуль через Админ -> Системный модуль -> Модули.

Подробные инструкции по установке модулей доступны в Руководстве по эксплуатации XOOPS

ИНСТРУКЦИИ ПО ЭКСПЛУАТАЦИИ

Чтобы настроить этот модуль, вам необходимо:

  • Вы можете войти в свой почтовый ящик прямо с главной страницы или щелкнув верхнюю ссылку «Перейти к модулю» в области администрирования ЛС.
  • Настройте свои предпочтения для модуля (см. «Предпочтения») и при необходимости блок ЛС, если вы собираетесь его использовать (см. «Блоки»).
  • Вы можете выполнить массовое удаление своих сообщений, используя вкладку «Удалить сообщения».

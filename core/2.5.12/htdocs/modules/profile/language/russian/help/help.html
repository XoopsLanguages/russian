Справка: Профиль

ОПИСАНИЕ

Модуль "Профиль" предназначен для управления настраиваемыми полями профиля пользователя.

УСТАНОВКА/УДАЛЕНИЕ

Никаких специальных мер не требуется, следуйте стандартному процессу установки – извлеките папку /profile в каталог ../modules. Установите модуль через Админ -> Системный модуль -> Модули.

Подробные инструкции по установке модулей доступны в Руководстве по эксплуатации XOOPS

ИНСТРУКЦИИ ПО ЭКСПЛУАТАЦИИ

Чтобы настроить этот модуль, вам необходимо:

  • Настройте свои предпочтения для модуля (см. «Предпочтения») и, при необходимости, блок «Профиль», если вы собираетесь его использовать (см. «Блоки»).
  • Отредактируйте существующие Категории или добавьте новые.
  • Отредактируйте существующие Поля или добавьте новые. Здесь вы можете указать, какие поля будут видны в какой категории и будут ли они видны при регистрации пользователя.
  • Определите порядок шагов Регистрации.
  • И, наконец, вы можете установить разрешения для отдельных полей - какие из них можно редактировать, по каким можно выполнять поиск.

Меры защиты от спама

Чтобы минимизировать регистрацию спама, сделайте следующее:

  • перейдите в модуль Protector в Админке, перейдите в Настройки и затем внизу, в этой опции: "Остановить Спам Форума" Проверяет данные POST на наличие спамеров, зарегистрированных в базе данных www.stopforumspam.com.
    Установите значение "Забанить IP (без ограничений)"

  • в /class/captcha/config.php убедитесь, что для режима установлено значение "text":

    return $config = array(
    'disabled' => false, // Отключить CAPTCHA
    'mode' => 'text', // режим по умолчанию, вы можете выбрать 'text', 'image', 'recaptcha'(требуется ключ api)
    'name' => 'xoopscaptcha', // имя captcha
    'skipmember' => true, // Пропустить проверку CAPTCHA для участников
    'maxattempts' => 10, // Максимальное количество попыток для каждой сессии
    );

  • В модуле "Профиль" перейдите в Админку и на шаге "Базовый", расположенном по адресу:
    /modules/profile/admin/step.php?id=1
    установите для параметра "Сохранить после остановки" значение "Нет"


  • В Настройках профиля установите для параметра "Использовать Captcha после второго шага регистрации" значение "Да" (это значение по умолчанию)

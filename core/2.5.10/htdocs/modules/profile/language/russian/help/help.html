Помощь: Профиль

ОПИСАНИЕ:

Модуль Профиль предназначен для управления настраиваемыми полями профиля пользователя.

УСТАНОВКА/УДАЛЕНИЕ

Никаких специальных мер не требуется, следуйте стандартному процессу установки - извлеките папку /profile в директорию ../modules. Установить модуль через Администрирование -> Системные модули -> Модули.

Подробные инструкции по установке модулей приведены в XOOPS Operations Manual

РУКОВОДСТВО ПО ОБСЛУЖИВАНИЮ

Для настройки этого модуля необходимо:

  • Настройте свои настройки для модуля (см. "настройки") и, возможно, блока профиля, Если вы собираетесь его использовать (см. "Блоки").
  • Редактировать существующие категории или добавлять новые.
  • Редактировать существующие поля или добавлять новые. Здесь можно указать, какие поля будут видны в какой категории и будут ли они видны при регистрации пользователя.
  • Определите порядок шагов регистрации.
  • И, наконец, Вы можете установить права для отдельных полей - какие из них доступны для редактирования, какие - для поиска.

Меры по борьбе со спамом

Чтобы свести к минимуму регистрацию спама, выполните следующие действия:

  • перейти в модуль Protector в Панели управления, перейдите в Настройки, а потом внизу, в этой опции: "Остановить спам на форуме" Проверяет данные POST против спамеров, зарегистрированных в базе данных на www.stopforumspam.com.
    Установите его в "запретить IP (без ограничений)"

  • в /class/captcha/config.php, убедитесь, что режим установлен как "текст":

    return $config = array(
    'disabled' => false, // Disable CAPTCHA
    'mode' => 'text', // default mode, you can choose 'text', 'image', 'recaptcha'(requires api key)
    'name' => 'xoopscaptcha', // captcha name
    'skipmember' => true, // Skip CAPTCHA check for members
    'maxattempts' => 10, // Maximum attempts for each session
    );

  • В модуле Профиль перейдите в Панели управления и в основной шаг, расположенный по адресу:
    /modules/profile/admin/step.php?id=1
    установите "сохранить после остановки" в "нет"


  • В настройках профиля установите для параметра "использовать Captcha после второго шага регистрации" значение "да" (по умолчанию)

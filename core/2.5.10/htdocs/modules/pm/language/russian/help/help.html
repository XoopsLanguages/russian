Помощь: Личные сообщения

ОПИСАНИЕ:

Модуль ЛС предназначен для личных сообщений между пользователями

УСТАНОВКА/УДАЛЕНИЕ

Никаких специальных мер не требуется, следуйте стандартному процессу установки - извлеките и переместите папку /pm в дирректорию ../modules. Установить модуль через Администрирование -> Системные модули -> Модули.

Подробные инструкции по установке модулей приведены в XOOPS Operations Manual

РУКОВОДСТВО ПО ОБСЛУЖИВАНИЮ

Для настройки этого модуля необходимо:

  • Вы входите в свой почтовый ящик непосредственно с главной страницы, или нажав на верхнюю ссылку “Перейти к модулю” в админ части.
  • Настройте свои настройки для модуля (см. "настройки") и, возможно, блок ЛС, если Вы собираетесь его использовать (см. " Блоки”).
  • Вы можете сделать массовое удаление Ваших сообщений с помощью вкладки ”Очистить" сообщения".

РУКОВОДСТВО

Учебник будет в ближайшее время.

Установка XOOPS

Первоначальная установка

Предисловие:

XOOPS - это динамический, OO (Объектно-Ориентированный), основанный на открытом исходном коде скрипт портала, написанный на PHP. XOOPS поддерживает различные базы данных. XOOPS - это идеальный инструмент для создания небольших высоко-динамичных сообществ вэб-сайтов, внутренних порталов компании, корпоративных порталов, web-публикаторов и многого другого.

XOOPS выпускается на условиях GNU General Public License (GPL) и может свободно использоваться и модифицироваться. Он распространяется свободно до тех пор, пока вы соблюдаете условия распространения GPL.

Требования

  • Вэб-сервер (Apache, NGINX, IIS, etc)
  • PHP 5.3.9 или выше, рекомендуется 7.1
  • MySQL 5.1 или выше 5.6 рекомендуется (Смотрите примечания по MySQL 8 здесь.)

Перед установкой

  1. Правильно установите вэб-сервер, PHP и базу данных MySQL.
  2. Подготовьте базу данных для вашего XOOPS сайта, установите кодировку utf8.
  3. Создайте пользователя и предоставить ему доступ к базе данных.
  4. Сделайте каталоги и файлы доступными для записи: %s (Дайте директории /uploads/ и файлу mainfile.php права на запись.)
  5. Из соображений безопасности настоятельно рекомендуем переместить эти две папки /xoops_lib/ (для библиотек XOOPS) и /xoops_data/ (для данных XOOPS) за пределы корня сайта и измените имена папок: %s
  6. Создайте (если его еще нет) и сделайте каталоги доступными для записи: %s. (Дайте директории /xoops_data/ права на запись; Создайте (Если их нет) и дайте права на запись следующим папкам /xoops_data/configs/, /xoops_data/caches/, /xoops_data/caches/xoops_cache/, /xoops_data/caches/smarty_cache/ and /xoops_data/caches/smarty_compile/.)
  7. Включите в браузере куки и JavaScript.

Установка на локальном компьютере

Если Вы используете локальный компьютер для установки и тестирования, то убедитесь что выполнели все предыдущие условия. Если всё нужное ПО установлено, то скопируйте содержимое каталога htdocs (из архива установки XOOPS) в корневой каталог Вашего сайта. Для начала установки введите в браузере ссылку на Ваш сайт, например "http://yoursite.com". Это запустит процесс установки.

Установка на платформе хостинга

Распакуйте архив установки XOOPS на Вашем локальном компьютере, или прямо на сервере, если у вас есть SSH доступ. Далее переместите все файлы из директории htdocs в корвеной каталог Вашего сайта. Для начала установки введите в браузере ссылку на Ваш сайт, например "http://yoursite.com". Это запустит процесс установки.
